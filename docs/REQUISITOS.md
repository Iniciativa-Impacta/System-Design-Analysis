# Requisitos do Sistema de Biblioteca

## Requisitos Funcionais

### Bibliotecário
- RF-BIB-01: Cadastrar novos livros no sistema rapidamente.
- RF-BIB-02: Registrar devoluções de livros e atualizar o inventário automaticamente.
- RF-BIB-03: Gerar relatórios de empréstimos por período.
- RF-BIB-04: Gerenciar multas automaticamente.

### Estudante
- RF-EST-01: Renovar empréstimos online.
- RF-EST-02: Receber notificações por e-mail sobre datas de devolução.
- RF-EST-03: Pesquisar livros por título, autor ou categoria.
- RF-EST-04: Reservar livros que estão emprestados.

### Professor / Pesquisador
- RF-PROF-01: Sugerir aquisição de novos livros.
- RF-PROF-02: Acessar relatórios de histórico de empréstimos para análise de tendências.

### Administrador
- RF-ADM-01: Controlar o acesso dos usuários ao sistema.
- RF-ADM-02: Criar diferentes perfis de usuário e gerenciar permissões.

### Visitante / Público
- RF-VIS-01: Consultar catálogo da biblioteca sem precisar de login.
- RF-VIS-02: Visualizar eventos e novidades da biblioteca.

## Requisitos Não Funcionais

- RNF-01 (Segurança): O sistema deve ser seguro e proteger os dados dos usuários.
- RNF-02 (Disponibilidade): O sistema deve ter alta disponibilidade de acesso (>99%).
- RNF-03 (Desempenho): O sistema deve responder a pesquisas em menos de 10 segundos.
- RNF-04 (Usabilidade): O sistema deve ter interface amigável e intuitiva.
- RNF-05 (Compatibilidade): O sistema deve ser compatível com dispositivos móveis.
